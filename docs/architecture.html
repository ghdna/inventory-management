Overview

What it is

A full-stack factory inventory management demo. Vue 3 frontend communicates with a Python FastAPI backend. All data is loaded from JSON files into memory at startup — no database required.

Key capabilities

Inventory tracking, order management, demand forecasting, spending analytics, backlog management, and quarterly reporting — all filterable by warehouse, category, time period, and order status.

Design goals

Minimal dependencies. No state management library (Pinia/Vuex). No ORM or database. Composables handle shared state. Demonstrates production-grade patterns in a self-contained demo.

Data Flow

1 — User
Filter Interaction
User selects values in FilterBar.vue — Period, Location, Category, or Status.
2 — Composable
useFilters.js
Singleton reactive state updates. selectedPeriod, selectedLocation, etc. Components watch() these refs.
3 — View
Vue Component
Watcher fires. Component calls api.getXX(getCurrentFilters()) with fresh filter object.
4 — API Client
api.js
Builds URLSearchParams from filters. Omits "all" values. Issues axios.get() with query string.
5 — Backend
FastAPI + main.py
apply_filters() runs sequential list comprehensions: warehouse → category → status → month.
6 — UI Update
Reactive Render
Response updates ref() data. Vue re-renders tables, charts, and KPIs via computed() properties.

Filter System

4 global filters flow from UI through composable state, API layer, to backend list comprehensions.
Filter Parameters
UI Label Composable ref Query param Values
Time Period selectedPeriod month all, 2025-01 … 2025-12, Q1–Q4
Location selectedLocation warehouse all, San Francisco, London, Tokyo
Category selectedCategory category all, Circuit Boards, Sensors, Actuators…
Order Status selectedStatus status all, Delivered, Shipped, Processing, Backordered
Backend Filtering Logic
# 1. warehouse filter
if warehouse != "all":
  items = [i for i in items
    if i.warehouse == warehouse]

# 2. category filter (case-insensitive)
if category != "all":
  items = [i for i in items
    if i.category.lower() == category.lower()]

# 3. month supports quarters (Q1-2025)
filter_by_month(items, month)
Note: Inventory endpoint does not support the month filter (no time dimension in stock data). Demand, Backlog, Spending, and Reports endpoints ignore all filters and return full datasets.
